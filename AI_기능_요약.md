# 🎉 새로 추가된 AI 기능 요약

리스크 관리 시스템에 **C. 진행 상황 자동 요약**과 **E. 자동 우선순위 정렬** 기능이 추가되었습니다!

---

## ✨ 추가된 기능

### 1️⃣ 진행 상황 자동 요약 (Progress Analysis)

**버튼 위치**: 상단 툴바 → "📊 진행 상황 분석"

**기능**:
- 전체 프로젝트 통계 자동 집계
- 완료율, 진행률 계산
- 지연된 작업 자동 감지
- 임박한 마감일 알림
- 우선순위별 현황 분석
- **AI 인사이트**:
  - 잘 진행되고 있는 점
  - 우려되는 점
  - 개선 제안
  - 집중해야 할 영역

**결과물**: 마크다운 형식의 상세 보고서

---

### 2️⃣ 자동 우선순위 정렬 (Priority Sorting)

**버튼 위치**: 상단 툴바 → "🎯 우선순위 정렬"

**기능**:
- 작업 긴급도 자동 계산 (마감일 기반)
- 작업 중요도 자동 평가 (체크리스트 우선순위 기반)
- 의존성 분석 (하위 작업 개수)
- 미완료율 고려
- **AI 기반 최적 순서 제안**
- 우선순위 레벨 자동 부여 (Critical/High/Medium/Low)

**결과물**:
- 우선순위 분석 보고서
- 작업 재정렬 (선택 가능)

## 🚀 사용 방법

### 기본 사용 (AI 없이)
```bash
python src/main.py
```
- 기본 통계 및 수치 기반 분석만 제공
- 가장 빠르고 간편함

### Ollama 사용 (무료 + 로컬 AI)
```bash
# 1. Ollama 설치: https://ollama.com/download
# 2. 모델 다운로드
ollama pull qwen2.5:7b

# 3. 실행
set LLM_MODE=ollama
python src/main.py

# 또는
실행_AI_Ollama.bat
```

### OpenAI 사용 (유료 + 정확)
```bash
# 1. API 키 발급: https://platform.openai.com
# 2. 실행
set OPENAI_API_KEY=sk-your-key
set LLM_MODE=openai
python src/main.py

# 또는
실행_AI_OpenAI.bat
```

## 💡 핵심 장점

### 1. 자동화된 프로젝트 모니터링
- 수동으로 통계 계산할 필요 없음
- 한 번의 클릭으로 전체 현황 파악
- 놓치기 쉬운 지연 작업 자동 감지

### 2. 스마트한 우선순위 관리
- 마감일, 중요도, 의존성 자동 고려
- AI가 최적의 작업 순서 제안
- 효율적인 시간 관리

### 3. 실행 가능한 인사이트
- 단순 통계를 넘어선 실질적 조언
- 개선 제안 자동 생성
- 집중해야 할 영역 명확화

### 4. 유연한 AI 모드 선택
- AI 없이도 사용 가능 (기본 기능)
- 무료 로컬 AI (Ollama)
- 고급 클라우드 AI (OpenAI)
- 상황에 맞게 선택 가능!

## 🔍 기술 상세

### 우선순위 계산 공식
```
우선순위 점수 = 긴급도(40%) + 중요도(30%) + 의존성(20%) + 미완료율(10%)
```

### 긴급도 계산
- 마감일 지남: 100점
- D-1: 95점
- D-3: 85점
- D-7: 70점
- D-14: 50점
- D-30: 30점

### 중요도 계산
- Critical 체크리스트: 100점
- High 체크리스트: 75점
- Medium 체크리스트: 50점
- Low 체크리스트: 25점

## 🆘 자주 묻는 질문

**Q: AI 없이도 사용할 수 있나요?**
A: 네! 기본 통계와 수치 기반 분석은 AI 없이도 작동합니다.

**Q: Ollama와 OpenAI 중 어떤 게 더 좋나요?**
A:
- **무료 + 보안 중시** → Ollama
- **정확도 + 속도 중시** → OpenAI

**Q: 분석 결과를 저장할 수 있나요?**
A: "📋 클립보드에 복사" 버튼으로 복사 후 원하는 곳에 붙여넣으세요.

**Q: 우선순위 정렬을 취소할 수 있나요?**
A: 정렬 적용 후 "⟲ 실행취소" (Ctrl+Z) 버튼을 누르세요.

**Q: 마감일은 어떻게 설정하나요?**
A: 각 토글의 `deadline` 필드에 YYYY-MM-DD 형식으로 입력하세요.
