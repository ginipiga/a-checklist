# 리스크 관리 시스템

노션과 같은 무한 중첩 토글 시스템을 제공하는 리스크 관리 애플리케이션입니다.

## 주요 기능

- **무한 중첩 토글**: 노션처럼 무제한 하위 토글 생성 가능
- **리스크 점수 관리**: 각 토글마다 현재/최대 점수 설정
- **자동 집계**: 상위 토글이 모든 하위 토글 점수 자동 합산
- **진행률 표시**: "17/25 완료" 형식으로 진행률 표시
- **다크 테마**: 깔끔한 다크 테마 UI
- **자동 저장**: 60초마다 자동 저장
- **Excel 내보내기**: 전체 구조를 Excel로 내보내기

## 설치 및 실행

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
cd src
python main.py
```

## 사용법

1. **새 토글 추가**: "새 루트 토글 추가" 버튼 또는 토글의 메뉴(⋮)에서 "하위 토글 추가"
2. **제목 편집**: 토글 제목을 클릭하여 인라인 편집
3. **점수 설정**: 현재 점수와 최대 점수를 spinbox로 설정
4. **내용 입력**: 상세 내용을 텍스트 영역에 입력
5. **토글 접기/펼치기**: ▼/▶ 버튼으로 하위 토글 표시/숨김
6. **삭제**: 토글 메뉴(⋮)에서 삭제 (루트 토글은 삭제 불가)

## 기술 스택

- **PyQt5**: GUI 프레임워크
- **openpyxl**: Excel 파일 생성
- **JSON**: 프로젝트 저장/불러오기

## 메뉴

- **파일**
  - 새 프로젝트 (Ctrl+N)
  - 프로젝트 열기 (Ctrl+O)
  - 저장 (Ctrl+S)
  - 다른 이름으로 저장 (Ctrl+Shift+S)
  - Excel로 내보내기
  - 종료 (Ctrl+Q)
